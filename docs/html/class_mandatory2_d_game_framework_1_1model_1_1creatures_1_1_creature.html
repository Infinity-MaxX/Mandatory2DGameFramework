Detailed Description

Represents a creature in the game world and serves as the base class for all creature types. Implements Template Method, Observer, and Strategy design patterns.

Template Method: The PerformHit method defines the attack flow, while subclasses may override AfterHit to customize post‑attack behavior.

Strategy: Damage calculation is delegated to an IHitStrategy implementation, allowing creatures to change combat behavior dynamically.

Observer: External observers can subscribe to hit and death events using RegisterObserver and RemoveObserver.

int Mandatory2DGameFramework.model.creatures.Creature.PerformHit ( Creature target)

Performs an attack on the specified target using the Template Method pattern.

Parameters
targetThe creature being attacked.
Returns
The amount of damage dealt.

This method defines the attack sequence:

  1. Invoke a customizable hook (BeforeHit).
  2. Calculate damage using the active hit strategy.
  3. Apply the damage to the target.
  4. Invoke a customizable hook (AfterHit).
