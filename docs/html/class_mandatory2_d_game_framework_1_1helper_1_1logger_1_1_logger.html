Detailed Description

Provides a thread-safe, singleton logger for writing trace information, warnings, and errors to registered listeners.

The Logger class does not add any default listeners. User must add one or more TraceListener instances using the AddListener method to receive log output. This design allows the logging framework to remain neutral and adaptable to different logging targets. The singleton instance can be accessed via the Log property.
