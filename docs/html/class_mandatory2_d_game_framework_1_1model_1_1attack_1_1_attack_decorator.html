Detailed Description

Represents a base decorator for attack items, allowing additional behavior to be layered on top of an existing AttackItem instance.

This class implements the Decorator Design Pattern. It wraps another AttackItem and forwards all property calls to the wrapped item unless overridden by subclasses. Concrete decorators can modify Hit, Range, Weight, or other behavior without changing the underlying item.
