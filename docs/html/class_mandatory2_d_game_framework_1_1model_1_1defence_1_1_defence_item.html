Detailed Description

Represents an object in the world that provides defensive capabilities by reducing incoming hit points.

This class implements the Composite design pattern, allowing multiple defence items to be treated as a single item. The composite sums the ReduceDamage and Weight values of all contained items. It can be used by creatures to represent carrying or equipping multiple defence items at once.
