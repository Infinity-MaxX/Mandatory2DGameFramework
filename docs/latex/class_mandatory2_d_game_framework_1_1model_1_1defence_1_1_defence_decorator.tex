\doxysubsection{Detailed Description}
Represents a base decorator for defence items, allowing additional behavior to be layered on top of an existing \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_item}{Defence\+Item} instance. 

This class implements the Decorator Design Pattern. It wraps another \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_item}{Defence\+Item} and forwards all property calls to the wrapped item unless overridden by subclasses. Concrete decorators can modify \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_decorator_a917713af7125e4c222a7a943af0cd9cc}{Weight}, \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_decorator_a7b9186d2e96f90e1f38021b06e4866e3}{Reduce\+Damage},or other behavior without changing the underlying item.
