\doxysubsection{Detailed Description}
Represents a composite attack item that groups multiple \doxylink{class_mandatory2_d_game_framework_1_1model_1_1attack_1_1_attack_item}{Attack\+Item} instances into a single logical weapon. 

This class implements the Composite design pattern, allowing multiple attack items to be treated as a single item. The composite sums the Hit, Range, and Weight values of all contained items. It can be used by creatures to represent carrying or equipping multiple weapons at once.
