\doxysubsection{Detailed Description}
Represents an object in the world that provides defensive capabilities by reducing incoming hit points. Implements operator overloading to allow combining multiple defence items, and serves as the base class for more specialized defence items that implements design patterns such as Decorator and Composite. 

This class implements the Composite design pattern, allowing multiple defence items to be treated as a single item. The composite sums the \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_item_a0daed8af64f17e3f2390a88b7e76c7eb}{Weight} and \doxylink{class_mandatory2_d_game_framework_1_1model_1_1defence_1_1_defence_item_aff658980d00ea677a411e11b50872825}{Reduce\+Damage} values of all contained items. It can be used by creatures to represent carrying or equipping multiple defence items at once.
