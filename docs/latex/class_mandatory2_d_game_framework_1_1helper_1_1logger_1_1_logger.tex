\doxysubsection{Detailed Description}
Provides a thread-\/safe, singleton logger for writing trace information, warnings, and errors to registered listeners. 

The \doxylink{class_mandatory2_d_game_framework_1_1helper_1_1logger_1_1_logger}{Logger} class does not add any default listeners. User must add one or more Trace\+Listener instances using the \doxylink{class_mandatory2_d_game_framework_1_1helper_1_1logger_1_1_logger_a543d8c665d9480d6c6c431ad5316a693}{Add\+Listener} method to receive log output. This design allows the logging framework to remain neutral and adaptable to different logging targets. The singleton instance can be accessed via the \doxylink{class_mandatory2_d_game_framework_1_1helper_1_1logger_1_1_logger_a63bb41f3132e391f305c2e4b17c2dd17}{Log} property.
