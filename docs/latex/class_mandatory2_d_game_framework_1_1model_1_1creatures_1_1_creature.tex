\doxysubsection{Detailed Description}
Represents a creature in the game world and serves as the base class for all creature types. Implements Template, Observer, and Strategy design patterns. 

{\bfseries{Template Method:\+}} The \doxylink{class_mandatory2_d_game_framework_1_1model_1_1creatures_1_1_creature_a4d37e9d85da0188fd103ac41fae577c4}{Perform\+Hit} method defines the attack flow, while subclasses may override \doxylink{class_mandatory2_d_game_framework_1_1model_1_1creatures_1_1_creature_a51d9c8de951060b3c1249705a331be4e}{After\+Hit} to customize post‑attack behavior. 

{\bfseries{Strategy:\+}} Damage calculation is delegated to an \doxylink{interface_mandatory2_d_game_framework_1_1game_interface_1_1strategy_1_1_i_strategy}{IStrategy} implementation, allowing creatures to change combat behavior dynamically. 

{\bfseries{Observer:\+}} External observers can subscribe to hit and death events using \doxylink{class_mandatory2_d_game_framework_1_1model_1_1creatures_1_1_creature_a83b429ea45feb959cfab975cb6b77de4}{Register\+Observer} and \doxylink{class_mandatory2_d_game_framework_1_1model_1_1creatures_1_1_creature_aef7d906a8465b33764596752b2e8dd69}{Remove\+Observer}.
