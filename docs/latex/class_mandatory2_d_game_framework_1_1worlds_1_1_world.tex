\doxysubsection{Detailed Description}
Represents the game world, including its dimensions, difficulty level, creatures and objects placed in the world. 

The \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world}{World} class serves as the primary container for game state, defining the playable area\textquotesingle{}s boundaries and difficulty. Use the \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world_aaea96f12a7b8707c491f8870f41ab04a}{From\+Config} method to create a \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world}{World} instance from a configuration object. The \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world_a0b1ad1e842a47530036cb945367b8465}{MaxX} and \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world_a448ca41211652392fcc724b3f06427d7}{MaxY} properties specify the world\textquotesingle{}s size along the X and Y axes, respectively. The \doxylink{class_mandatory2_d_game_framework_1_1worlds_1_1_world_ae47b636a19335bfbbbdaed97a7ede666}{Difficulty} property determines the overall challenge level for the world.
